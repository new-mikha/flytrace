- All of the stuff below is OPTIONAL. You don't need it for the tracking, it's just a service feature.
- Event is e.g. a competition or fly-in.
- Each event has a list of waypoints, it's the same list as pilots upload to their varios before the comp.
- Event's waypoints can be used to set up a Today Task for the event.
- You're supposed to change event's Today Task at every day of the comp.
- A Pilot Group could have an event assigned, in this case the group's map will display the Today Task for the event.
- In Single Task Mode there is only one default event, i.e. in this mode you have only one Today Task for all your groups.
- In Multiple Events Mode you can have many events. E.g. you could use that if you have a joint competition with more than one task running together.
